Dear $firstName,

Welcome to the Wycliffe Web Services Command processor. You can email a list of web service calls according to the template between the horizontal lines, or attach a command file named sample.csv in a format similar to this sample. The result of the last web service call will be emailed back to you. And because it's an email, you can forward the email from your sent box to perform the same command again.

You can also try out the web interface, or read the full documentation.

The following sample web call will call the email web service. Just set the fromName and to parameters to your own email. Text outside of the horizontal lines will be ignored by the email processor, except for the simulate parameter.

URL: http://wycliffe-services.net/email/webservice.php
fromName: Your name
from: webservice@wycliffe-services.net
to: your_email@email.com
subject: Sent from the Wycliffe Web Services Command web service
body:
Dear $fromName,

You can put "anything" you like here.

From,
Me
/body: # The text between body: and /body: will be taken as the body parameter. The # sign is a comment, and will ignore everything after it.
# attach1: _file1 # The first file attached to this email will be sent to the Email web service as attach1 if you uncomment this line
simulate: 0 # Set simulate = 1 to run through all the checks but not actually process the commands
